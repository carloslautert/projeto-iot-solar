# Observações de entrega

Este repositório foi preparado para entrega acadêmica e publicação no GitHub.

## Segurança

O projeto original continha credenciais do TagoIO em arquivos locais. Na versão final, o token foi removido do código-fonte e deve ser informado apenas no arquivo `.env`, que está listado no `.gitignore` e não deve ser enviado ao GitHub.

Caso algum token real já tenha sido compartilhado em arquivo, print, vídeo ou repositório, gere um novo token no TagoIO e substitua o antigo no arquivo `.env` local.

## Arquivos auxiliares originais

O ZIP original também continha roteiros de vídeo em DOCX/PDF e um arquivo `.bat`. Esses arquivos foram analisados, mas não foram incluídos na estrutura final do repositório para evitar exposição desnecessária de dados pessoais ou informações de conta. Para o Moodle, eles podem ser anexados separadamente, se o professor tiver solicitado.

## Relatório

O relatório foi entregue em duas versões: `docs/relatorio.tex`, para edição/compilação em LaTeX, e `docs/relatorio.pdf`, já compilado para anexar diretamente no Moodle.

## Dashboard

A pasta `assets/imagens_do_dashboard/` foi deixada preparada para incluir capturas de tela do dashboard do TagoIO, se desejado.
